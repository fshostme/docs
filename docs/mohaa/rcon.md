### Medal of Honor: Allied Assault: Rcon Commands

- rconpassword [rconpass]<br />
To login as admin

- cmdlist<br />
list all console commands

- rcon say [message]<br />
send message to all players in the game

- rcon quit<br />
kills the server

- rcon map [map name]<br />
change the map

- rcon kick all<br />
Kicks everyone off the server at once
